# ToolKit Pro ⚡

> 15+ free online tools for students, developers and professionals.
> No sign-up. No ads. No limits. 100% browser-based.

Live Site: https://creativestack-web.github.io/toolkit-pro/

---

## Tools Available

- 📏 Unit Converter — Length, weight, temp, speed, area, data
- 💱 Currency Converter — 9 major world currencies
- 🧮 Scientific Calculator — Trig, logs, powers, memory
- 📷 QR Code Generator — URL, text, email, WiFi
- 🏦 EMI Calculator — Monthly EMI + amortization schedule
- 💪 BMI Calculator — Metric and imperial, WHO categories
- 🎂 Age Calculator — Exact age + next birthday
- 💯 Percentage Calculator — 3 modes, instant results
- 🏷️ Discount Calculator — Sale price + savings
- 🎓 GPA Calculator — 4.0 scale, multi-subject
- 🔐 Password Generator — Strength meter + custom options
- 📝 Word Counter — Words, characters, read time
- 🔤 Text Case Converter — 6 formats including camelCase
- 🎲 Random Number Generator — Custom range + unique sets
- ⏱️ Stopwatch and Timer — Laps + countdown timer

- Instant Results — Updates as you type
